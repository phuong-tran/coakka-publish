## Runtime Changes

- Internal allocator behavior now normalizes zero-size allocation requests and
  rejects overflow in zero-filled allocation paths.
- Layout-sensitive runtime code uses a conservative cache-line fallback with
  explicit handling for common 64-byte platforms and 128-byte `ppc64`.
- The public native package includes the runtime C ABI and scanner-clean TCP
  frame transport candidate for the supported public platforms.
- The public sample set includes route hot-reload coverage and smoke-load
  benchmark harnesses. macOS numbers are reference-only development smoke data;
  Linux CI/hardware evidence should be used before making broader performance
  claims.
