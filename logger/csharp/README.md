# logger/csharp

This lane stores binary-only C#/.NET logger package drops.

Published releases live under:

```text
logger/csharp/releases/<native-package-version>/
```

Each release directory contains:

- `CoAkka.Logger.<version>.nupkg`
- `manifest.json`
- `SHA256SUMS`
- `README.md`
- `CONSUMING.md`
