# CoAkka Runtime Native v2

This lane publishes the direct runtime v2 native C ABI archive.

Current release:

```text
runtime/native/releases/0.1.0+63c346e/coakka-runtime-native-v2-0.1.0.tar.gz
```

The package is local/runtime-only. Runtime JVM, Python, Node.js, Go, C#, Rust,
Spring Boot, and Quarkus artifacts are published against this native package
version.
